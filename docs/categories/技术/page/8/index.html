安卓手机修改主机名

 2015-02-02  751 字  android 

本文年代久远，可能已经不适用。仅供参考！

总有一些强迫症的奇葩，看到安卓手机在路由器里面的主机名称示诸如android-515------**23b22之类的名称，感到不爽，非常不方便识别自己的手机。

一般情况下，安卓手机的主机名为类似于android-515------**23b22这样的字符串，在路由器下的显示不够友好，不方便管理。针对root过的手机，可以按照本文的方法修改主机名。
