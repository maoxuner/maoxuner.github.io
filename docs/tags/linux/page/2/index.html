日常使用ArchLinux

折腾了一阵子MacOS，发现它的软件管理太麻烦了：

  • AppStore（系统更新、部分软件包）
  • 网络上的软件包
  • HomeBrew（GNU、Casks）

大部分Casks不能更换软件源，更新速度极慢，加上Mac机器老旧，新上市的AMD yes等原因，便回到了linux阵营。
